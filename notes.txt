4.. Get Request

Await is a mandatory keyword when you trigger functions wich are promises.
	Promise is the type of function in js that should return you the result within a defined timeout
	Promose has just two status resolve or rejected
and by default playwright will wait for 30 sec to execute the function


To build the TC:

2. Add the parameters

test('Get test Articles', async ({request}) => {
  // Create the request and get the response
  const articlesResponse = await request.get('https://conduit-api.bondaracademy.com/api/articles')

  // Extract the JSON body
  const articlesResponseBodyJSON = await articlesResponse.json()

});

5. Assertions
test('Get test Articles', async ({request}) => {
  // Create the request and get the response
  const articlesResponse = await request.get('https://conduit-api.bondaracademy.com/api/articles')

  // Extract the JSON body
  const articlesResponseBodyJSON = await articlesResponse.json()

  expect(articlesResponse.status()).toEqual(200)
  expect(articlesResponseBodyJSON.articlesCount).toEqual(10)
});
